# 🚀 AI-Powered Loan Approval Automation

## 📌 Overview
AI-Powered Loan Approval Automation is a workflow automation project developed using **n8n** and **Google Sheets** to streamline the loan approval process. The system automatically evaluates loan applications based on predefined eligibility criteria and classifies them as **Approved** or **Rejected**, reducing manual effort and improving decision-making efficiency.

---

## 🎯 Objectives
- Automate loan application screening.
- Reduce processing time and human errors.
- Improve consistency in approval decisions.
- Demonstrate the use of workflow automation in finance.
- Create a scalable solution for loan management.

---

## 🛠️ Technology Stack
- **n8n** – Workflow Automation
- **Google Sheets** – Data Storage
- **GitHub** – Version Control & Collaboration

---

## 📊 Workflow Description
1. Loan application details are entered into Google Sheets.
2. n8n fetches applicant data automatically.
3. The workflow evaluates key financial parameters:
   - Credit Score
   - Monthly Income
   - Existing Debt
   - Employment Status
4. Decision logic is applied.
5. Loan status is generated:
   - ✅ Approved
   - ❌ Rejected
6. Results are stored and updated automatically.

## ⚙️ Sample Approval Criteria

| Parameter | Requirement |
|------------|------------|
| Credit Score | Above 700 |
| Monthly Income | Above ₹50,000 |
| Debt-to-Income Ratio | Below 40% |
| Employment Status | Employed |

Applications meeting all criteria are approved automatically.
